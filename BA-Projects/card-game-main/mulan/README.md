# Card Game

## Overview
This project was completed during my Bachelor’s degree studies. The goal was to create a card game using C++.

The project used a simple graphics library provided by the professor. My task was to use that library to design and implement the game logic and functionality.

## Technologies Used
- C++
- Simple graphics library provided by the professor
- Object-oriented programming
- Basic graphics
- Game development concepts

## Main Features
- Card game implementation
- Basic graphical interface
- User interaction
- Game rules and logic
- Class-based C++ structure

## What I Learned
- How to build a game using C++
- How to work with a provided graphics library
- How to design game logic
- How to organize code using classes and functions
- How to connect program logic with simple graphics

## Academic Context
This project was completed as part of my BA studies.
