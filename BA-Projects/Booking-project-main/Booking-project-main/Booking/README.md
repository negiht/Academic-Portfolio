# Theater Booking Project

## Overview
This project was completed during my Bachelor’s degree studies for the Operating Systems course.

The goal was to create a theater booking application using C. The project used threads to simulate multiple users trying to book seats at the same time.

## Project Purpose
The purpose of this project was to understand how operating systems manage concurrency and how threads can be used in a real application scenario.

## Technologies Used
- C
- Threads
- Operating systems concepts
- Concurrency
- Synchronization

## Main Features
- Theater seat booking system
- Multithreaded execution
- Simulation of multiple customers booking at the same time
- Seat availability management
- Basic synchronization logic

## What I Learned
- How threads work in C
- How concurrency problems appear in real applications
- How to manage shared resources
- How operating systems handle simultaneous tasks
- How to structure a C project

## Academic Context
This project was completed as part of my BA studies in the Operating Systems course.
