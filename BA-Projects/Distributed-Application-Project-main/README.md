# Distributed Application Project

## Overview
This project was completed during my Bachelor’s degree studies. The goal was to create an activity tracker using Java, based on a distributed architecture with a master-worker model.

The system used multithreading to allow different parts of the application to run concurrently. It also included an Android application component, which acted as the user-facing app for the activity tracker.

## Project Purpose
The purpose of this project was to design and implement a distributed activity tracking system where tasks could be handled by multiple workers under the coordination of a master component.

## Technologies Used
- Java
- Android
- Multithreading
- Master-worker architecture
- Distributed systems concepts

## Main Features
- Activity tracking system
- Master-worker structure
- Multithreaded execution
- Android app component
- Communication between distributed parts of the system
