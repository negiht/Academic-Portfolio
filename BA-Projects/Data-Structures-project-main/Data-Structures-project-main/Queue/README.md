# Data Structures Project

## Overview
This project was completed during my Bachelor’s degree studies for the Data Structures course.

The project focused on understanding and implementing queue-based data structures in Java. It included exercises related to queues, double-ended queues, nodes, palindrome checking, and expression conversion.

## Project Purpose
The purpose of this project was to practise core data structures and learn how queues can be implemented and used in different algorithmic problems.

## Technologies Used
- Java
- Data structures
- Queue
- Double-ended queue
- Object-oriented programming

## Main Features
- Queue implementation
- Double-ended queue implementation
- Node-based structure
- DNA palindrome checking
- Postfix to infix expression conversion

## What I Learned
- How queues work internally
- How to implement data structures using Java
- How nodes are used in linked structures
- How to solve algorithmic problems with queues
- How to organize Java code into packages and classes

## Academic Context
This project was completed as part of my BA studies in the Data Structures course.
