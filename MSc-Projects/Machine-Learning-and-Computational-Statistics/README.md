# Machine Learning and Computational Statistics

## Overview
This folder contains assignments and projects completed for the Machine Learning and Computational Statistics subject during my MSc studies.

The subject focused on machine learning methods, statistical modelling, computational approaches, model evaluation, and the practical implementation of data-driven algorithms.

## Topics Covered
- Supervised learning
- Unsupervised learning
- Classification
- Regression
- Statistical learning
- Computational statistics
- Model evaluation
- Training and testing datasets
- Feature selection
- Overfitting and underfitting
- Predictive modelling

## Tools and Technologies
- Python
- Jupyter Notebook
- R
- RStudio
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Statistical modelling
