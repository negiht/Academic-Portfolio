# Large Scale Data Management

## Overview
This folder contains assignments and projects completed for the Large Scale Data Management subject during my MSc studies.

The subject focused on the design, storage, processing, and management of large-scale data systems. It covered distributed file systems, distributed processing frameworks, NoSQL databases, and scalable storage architectures.

## Topics Covered
- Large-scale data systems
- Distributed storage
- HDFS
- Google File System (GFS)
- Apache Spark
- MapReduce concepts
- NoSQL databases
- DynamoDB
- Column stores
- Key-value stores
- Distributed databases
- Fault tolerance
- Replication
- Scalability
- Data partitioning

## Tools and Technologies

- Big data architectures
