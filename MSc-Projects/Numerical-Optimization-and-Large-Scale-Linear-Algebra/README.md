# Numerical Optimization and Large Scale Linear Algebra

## Overview
This folder contains assignments and projects completed for the Numerical Optimization and Large Scale Linear Algebra subject during my MSc studies.

The subject focused on mathematical and computational methods used to solve optimization problems and large-scale linear algebra problems efficiently.

## Topics Covered
- Numerical optimization
- Large-scale linear algebra
- Linear systems
- Matrix computations
- Eigenvalues and eigenvectors
- Iterative methods
- Gradient-based optimization
- Convex optimization
- Least squares problems
- Numerical stability
- Computational efficiency
- Large-scale problem solving

## Tools and Technologies
- Python
- Jupyter Notebook
- NumPy
- SciPy
- Matplotlib
- Mathematical modelling
- Numerical methods
- Optimization algorithms
