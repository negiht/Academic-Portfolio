# Practical Data Science

## Overview
This folder contains assignments and projects completed for the Practical Data Science subject during my MSc studies.

The subject focused on applying data science techniques to real or practical datasets, including data preparation, analysis, visualization, and interpretation of results.

## Topics Covered
- Data cleaning
- Data preprocessing
- Exploratory data analysis
- Data visualization
- Statistical analysis
- Machine learning basics
- Python for data science
- Working with datasets

## Tools and Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- CSV datasets
