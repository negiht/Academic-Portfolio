# Assignment 3: Human Decision Making Under Uncertainty

## Overview
This assignment was completed for the Practical Data Science subject during my MSc studies.

The assignment was based on the choices13k dataset, which contains human decision rates on risky choice problems. The aim was to explore how people make decisions under uncertainty using a real research-based dataset.

## Source Dataset
The assignment used the **choices13k dataset**.

The dataset contains human decision rates on 13,006 risky choice problems. Each problem asks participants to choose between two risky options, also called gambles. The dataset is connected to research in psychology, economics, and machine learning.

## Project Purpose
The purpose of this assignment was to practise practical data science methods using a dataset related to human decision-making under risk and uncertainty.

## Dataset Description
The dataset includes decision problems where participants selected between two options, Gamble A and Gamble B. Each option involved rewards and probabilities.

Important dataset features include:
- Problem ID
- Number of participants
- Feedback condition
- Gamble B selection rate
- Reward values
- Probability values
- Ambiguity condition
- Correlation between gamble outcomes
