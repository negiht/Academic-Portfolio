# Assignment 1: Doughnut Economics Data Analysis

## Overview
This assignment was completed for the Practical Data Science subject during my MSc studies.

The assignment was based on a scientific paper and related repository about Doughnut Economics and social and planetary boundaries. The purpose was to explore the provided material, understand the dataset and analysis context, and work with the assignment notebook.

## Source Material
The assignment was based on the following publication/repository:

Fanning, A.L. and Raworth, K. (2025). *Doughnut of Social and Planetary Boundaries Monitors a World Out of Balance*, Nature.

The original repository README mentions that the analysis used R and several packages for data organisation, manipulation, visualisation, time-series analysis, statistical analysis, and data conversion.

## Project Purpose
The purpose of this assignment was to practise practical data science skills using real research-based data and analysis material.
