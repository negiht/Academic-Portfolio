# Machine Learning

## Overview
This folder contains assignments and projects completed for the Machine Learning subject during my MSc studies.

The subject focused on the main concepts and techniques used to build, train, evaluate, and interpret machine learning models.

## Topics Covered
- Supervised learning
- Unsupervised learning
- Classification
- Regression
- Model evaluation
- Feature engineering
- Training and testing datasets
- Overfitting and underfitting
- Machine learning algorithms
- Practical model implementation

## Tools and Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Data preprocessing
- Machine learning models
