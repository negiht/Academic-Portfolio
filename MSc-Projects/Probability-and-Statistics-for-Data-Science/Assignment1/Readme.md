# Assignment 1: Probability and Statistics for Data Science

## Overview
This assignment was completed for the Probability and Statistics for Data Science subject during my MSc studies.

The assignment focused on probability distributions, statistical estimation, descriptive statistics, confidence intervals, and statistical analysis using R.

## Project Purpose
The purpose of this assignment was to practise the main statistical and probabilistic concepts used in data science, both theoretically and through R programming exercises.

## Exercise 5
This exercise uses R to generate samples from:
- Poisson distribution with λ = 3
- Exponential distribution with rate = 1

The script compares samples of size 100 and 10,000, calculates descriptive statistics, creates plots, and compares empirical and theoretical quantiles.

## Exercise 6
This exercise uses R to analyze student grades and satisfaction data.

The script calculates:
- A 95% confidence interval for the mean Marketing score
- A 90% confidence interval for the mean Accounting score
- The percentage of satisfied students
- A 95% confidence interval for the satisfaction proportion
