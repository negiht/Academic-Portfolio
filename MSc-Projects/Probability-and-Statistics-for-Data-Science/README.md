# Probability and Statistics for Data Science

## Overview
This folder contains assignments and projects completed for the Probability and Statistics for Data Science subject during my MSc studies.

The subject focused on the statistical and probabilistic foundations needed for data science, including probability theory, descriptive statistics, inferential statistics, distributions, hypothesis testing, and statistical analysis.

## Topics Covered
- Probability theory
- Random variables
- Probability distributions
- Descriptive statistics
- Inferential statistics
- Hypothesis testing
- Confidence intervals
- Correlation and regression
- Statistical analysis for data science

## Tools and Technologies
- R
- RStudio
- Statistical analysis
- Data visualization in R
- Probability and statistics methods
