## Main Topics

### Exercise 1: Recall Ability and Experience Level
This exercise analysed whether recall ability differs between novice, advanced, and proficient players.

The analysis included:
- Descriptive statistics
- Boxplot visualization
- Shapiro-Wilk normality tests
- Levene’s test for equal variances
- Kruskal-Wallis test
- Dunn’s post-hoc test with Bonferroni adjustment

The results showed that experience level was significantly associated with recall ability.

### Exercise 2: Alcohol, Gender, and Attractiveness Ratings
This exercise examined whether alcohol consumption and gender affected attractiveness ratings.

The analysis included:
- Descriptive statistics
- Boxplots and interaction plots
- Two-way ANOVA with interaction
- Model assumption checks

The results showed a significant interaction between alcohol consumption and gender.

### Exercise 3: Reading Time Analysis
This exercise analysed reading time for general and technical documents.

The analysis included:
- Comparison of reading times between document types
- Exploratory visualizations
- Multiple linear regression
- Regression diagnostics
- Prediction intervals

The fitted regression model used word count, readability, image count, and document type to explain reading time.

### Exercise 4: Mortgage Denial Analysis
This exercise used mortgage application data to analyse factors related to mortgage denial.

The analysis included:
- Logistic regression
- AIC and BIC model selection
- Odds ratio interpretation

The results suggested that financial risk, credit history, and marital status were associated with mortgage denial.

### Exercise 5: Covid-19 Vaccination and Mortality
This exercise analysed the relationship between vaccination coverage and Covid-19 mortality across EU countries.

The analysis included:
- Exploratory analysis
- Linear regression
- Poisson regression
- Negative Binomial regression
- Model comparison using AIC

The results showed a strong negative association between vaccination coverage and Covid-19 mortality.
