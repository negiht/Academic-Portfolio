# Text Analytics

## Overview
This folder contains assignments and projects completed for the Text Analytics subject during my MSc studies.

The subject focused on Natural Language Processing, text representation, machine learning for text data, and deep learning models used for language understanding and text classification.

## Topics Covered
- Natural Language Processing
- Text preprocessing
- Tokenization
- Stopword removal
- Stemming and lemmatization
- Bag-of-Words
- TF-IDF
- Word embeddings
- Text classification
- Sentiment analysis
- Neural networks for text
- Multilayer Perceptrons
- Recurrent Neural Networks
- Convolutional Neural Networks
- Transformers
- Deep learning for NLP

## Tools and Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- NLTK / spaCy
- TensorFlow / Keras
- PyTorch
- Transformers
- Deep learning models
